## 4. 配置说明

### 环境变量 (.env)

| 变量 | 说明 | 获取方式 |
|------|------|----------|
| FEISHU_APP_ID | 飞书应用ID | 开发者后台 |
| FEISHU_APP_SECRET | 飞书应用密钥 | 开发者后台 |
| FEISHU_APP_TOKEN | 多维表格Base Token | 浏览器URL中 |
| FEISHU_TABLE_ID | 需求库表ID | 浏览器URL中 |
| FEISHU_TASK_TABLE_ID | 采集任务表ID | 浏览器URL中 |
| FEISHU_WEBHOOK_URL | 飞书群机器人URL | 群设置中添加机器人 |

### 飞书 Webhook 配置（推荐）

飞书个人版无法通过应用API写入多维表格，但可以通过群机器人Webhook推送通知。

**配置步骤：**
1. 打开飞书桌面客户端
2. 进入目标群聊 -> 右上角 ... -> 设置
3. 找到群机器人 -> 添加机器人 -> 自定义机器人
4. 设置名称 -> 添加 -> 复制Webhook URL
5. 打开 Dashboard -> Webhook页 -> 粘贴URL -> 保存 -> 测试连接

## 5. Dashboard 使用指南

### 总览
展示任务统计概览：待执行、运行中、已完成、失败的任务数量。

### 采集任务
显示从飞书采集任务表读取的任务列表。支持Dry-Run（预览命令）、启动、停止、新建、编辑操作。

### 本机直接采集
无需飞书任务表，直接填写参数启动采集。支持设置平台、采集类型、关键词、登录方式、保存格式等。

### 需求库 / 导出文件
展示已采集的数据文件，支持预览、下载、同步到飞书（需企业版）。

### 配置检查
显示所有环境变量状态，确认飞书连接是否正常。

### Webhook 配置
配置飞书群机器人通知，采集完成后自动推送结果。

## 6. 采集方式

### 本机采集 vs 飞书任务

| 特性 | 本机采集 | 飞书任务 |
|------|---------|---------|
| 配置存储 | 浏览器临时 | 飞书持久化 |
| 适用场景 | 临时测试 | 正式批量 |
| 飞书推送 | 支持Webhook | 支持Webhook |

采集完成后自动推送消息到飞书群，包含：采集时间、平台、关键词、统计、前5条内容摘要、输出文件路径。

## 9. 常见问题

### Q1: 采集时弹出二维码？
首次采集需要扫码登录，用手机App扫描即可。

### Q2: 为什么链接打不开？
小红书等平台使用反爬机制，链接有时效性。可在App内搜索标题验证数据真实性。

### Q3: 数据没有同步到飞书多维表格？
飞书个人版不支持应用API写入多维表格。请使用Webhook群机器人推送。
